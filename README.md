## 1. 介绍

一个简单的httpclient，基于Apache HttpClient

## 2. 其他

已上传Maven中央仓库，请在pom.xml中引用

``` xml
<!-- https://mvnrepository.com/artifact/xyz.migoo/simplehttp -->
<dependency>
    <groupId>xyz.migoo</groupId>
    <artifactId>simplehttp</artifactId>
    <version>${version}</version>
</dependency>
```
